# Parental Leave Application


**PARENTAL LEAVE APPLICATION** is a small school project built with **Next.js**, **React Hook Form** and **Zod**. It is a multi-step form consisting of 7 steps: applicant information, employment details, partner information, leave period, payment details, document upload and a final review and submit step.



## How to Run

1. Install dependencies and Start the developement server

|            | go to directory | install dependencies | run           |
| ---------- | --------------- | -------------------- | ------------- |
| run app    | `cd my-app`     | `npm install`        | `npm run dev` |

2. Open the app in your browser

```
http://localhost:3000
```

## Tech Stack

- Next.js
- React Hook Form
- Zod
- Tailwind CSS
